Task Manager Help

Welcome to the Task Manager! Here is the online help that should assist you with any of your needs:

How to add a main task?

Type your intended task name in the text box, then input the date of task submission and pick a color if desired for the task.

How to set the date?

You can either click the calendar symbol on the right of the date box or manually type the date.

How to set the color of a task?

Click the button below the task color subheading and select your desired color.

How to remove a task?

Click the remove button in the top right of the task container.

How to add a subtask?

Fill in the details and click the "Add Sub Task" button. The same rules apply for entering the data as main tasks.

How to delete all data?

Click the button in the top right of the page; a pop-up will confirm if you want to delete all data.
